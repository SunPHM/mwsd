LaTeX

Google News Group: comp.text.tex

A quick and easy way to get your TeX/LaTeX questions answered.

Hypertext Help with LaTeX

This site is hosted by the Goddard Institute for Space Studies, which is a division of NASA. It is a highly comprehensive on-line LaTeX help resource. The site is quite large but it is easy to navigate, as the table of contents can be ordered by subject, LaTeX command, or environment.

LaTeX Project

As the name suggests, The LaTeX Project is a specialist LaTeX site. It's the place to go for all the latest on LaTeX, in particular LaTeX3.

TeX Users Group

The TeX Users Group is the mother of all TeX/LaTeX sites. All things TeX/LaTeX can be downloaded (mostly for free) from this site — anything from documentation, packages, to LaTeX itself. The Comprehensive TeX Archive Network (CTAN) can also be accessed through this site.

The (Not So) Short Introduction to LaTeX2e

  • version 3.3 (852K)
  • version 4.13 (2.2M)

This is THE essential book for first time LaTeX users.

Miscellaneous

Set is a card game that tests your ability to see patterns. In this web version, the aim is to find six "sets" in the quickest time possible. My best time is 20 seconds, but my average would be about 1min 30sec. True set gurus frequently score around the 30 second mark, so I'm not that good. Warning: This game is highly addictive.

Star Trek is a bit of fantasy that I indulge myself in every week. OK, so I'm a geek, but you won't find me dressing up or going to Star Trek conferences ... I promise!

The Infamous Worm Game

This is similar to the "Snake Game" which can be found on many mobile phones. Your mission is to use your mouse to navigate your worm through obstacles and an undulating terrain. As you successfully complete the course your worm gets longer; you are awarded a point for every pixel. Sure, it's pretty lame, but once again it's highly addictive!!!

The name says it all. I find it more convenient than flicking through an actual dictionary. There is also a thesaurus.

Shell Script

Advanced Bash-Scripting Guide

This is a complete online book with hundreds of excellent, well explained BASH scripts. Don't be intimidated by the title, this guide assumes no prior knowledge of shell scripting.

Archive for "Using csh & tcsh"

You can obtain the CSH and TCSH manuals from this page in many different formats, including PS and PDF.

Bash Reference Manual

This is the official manual for the GNU command language interpreter. The BASH manual can be obtained in numerous formats, including PS and HTML.

How to write a shell script

A brief introduction to Bourne shell (SH) programming.

Linux Online - BASH Programming

This is another online book on BASH shell scripting, where the emphasis is on introduction.

Linux Shell Scripting Tutorial - A Beginner's handbook

Another comprehensive online book, primarily on BASH scripting. I haven't checked this one out in detail, but I thought another reference can't hurt.

Sed one-liners

Sed stands for Streaming EDitor. This essential shell scripting tool is great for editing text files on the fly. As the name suggests, this site has a one-line sed command for nearly every occasion.

CSH/TCSH Tutorial (PDF 96K)

This is a fabulous introductory tutorial on programming in the CSH/TCSH shells.

Template for Crane Simulation

Historically, the SPS techniques used in crane selection have made use of "relationship logic links". These, however, have been ineffective in adequately responding to the rapid increase in the number of work tasks required of the tower crane resource. As the techniques became less and less equipped to deal with the increasing demands, they became more unmanageable and, therefore, less appealing to the industry. The template that has been proposed to replace the traditional one, which made use of the older and less pragmatic techniques, employs "priority rating logic" rather than "relationship logic links". In a study done for the Electrical and Computer Engineering Research Facility (ECERF), the advantages of this representation were determined and analyzed.

This simulation model created with the SPS tower crane template (using "priority rating logic" control) required a number of inferences to be made during development. First, the parameters that are needed for each work package are representative of the tower crane construction domain, which results in a transference of knowledge that is clear and understandable for site personnel. Second, practitioners have available to them the Creating Site footprint tool, which will enable them to envision the actual construction layout. All of the required information is drawn directly from site drawings. Third, using "priority rating logic", the development phase of the tower crane template will be facilitated greatly and made significantly more efficient. This is because this control system uses work package priority ratings and arrival times to control logic in the modeled system. Fourth, scenario analysis by the practitioner is encouraged due to the model’s increased flexibility. Fifth, the overall environment becomes simpler to understand because the user has been isolated from the low level constructs and instead is presented with a model interface that more closely represents the actual tower crane system. These inferences were shown to be accurate by results exhibited by the ECERF SPS tower crane template, as it produced results similar to the common template approach. This demonstrated the possibility and viability of replacing the representation used by the common template, that is, "relationship logic links", with "priority rating logic control" for modeling process interaction.
